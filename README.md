# Celso & Karen — Invitación de boda

Sitio web de una sola página para la boda de **Celso Hernández & Karen Garrido**, el **10 de octubre de 2026**.

Diseño minimalista y elegante en tonos arena, crema y tierra. Construido con HTML + Tailwind CSS (vía CDN), 100% responsivo, en un único archivo autónomo (`index.html`).

## Secciones
- Hero a pantalla completa con los nombres y la fecha
- Logotipo oficial y mensaje de bienvenida
- Cuenta regresiva en vivo
- Nuestra historia
- Ceremonia y recepción + cronograma del evento
- Cómo llegar (auto, avión, hospedaje)
- Mesa de regalos
- Código de vestimenta + preguntas frecuentes
- Confirmación de asistencia por **WhatsApp** (+52 56 1111 1733)

## Publicar en GitHub Pages
1. Sube esta carpeta a un repositorio en GitHub.
2. En **Settings → Pages**, elige la rama `main` y carpeta `/ (root)`.
3. El archivo `CNAME` ya apunta a `karenycelso.com`. Configura en tu proveedor de dominio:
   - Registro `A` de `karenycelso.com` a las IPs de GitHub Pages, y
   - Registro `CNAME` de `www` a `<tu-usuario>.github.io`.
